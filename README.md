# TodoList in Java

A lightweight, terminal-based (CLI) To-Do List manager built in Java.

> [!WARNING]
> **Work in Progress:** This project is currently under construction and active development.

## Features (In Development)
- **Task Management:** Add, remove, and list tasks directly in the terminal.
- **Task Customization:** Update task names, adjust priorities, and toggle completion status.
- **File Parsing:** Load initial tasks from a text configuration file.

## Requirements & Run
- Java JDK 17 or higher
- To run the application, execute from the root directory:
  ```bash
  java -cp bin app.App
  ```
